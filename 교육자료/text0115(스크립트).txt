화살표함수 = 익명함수(즉시 실행이 필요할 경우 사용)

* 바인딩 : 함수 호출과 실제 함수를 연결하는 방법
	 함수를 호출하는 부분에서 실제 함수가 위치한 메모리를 연결하는 것
	 정적바인딩 - 실행 시간 전에 일어남. 실행 시간엔 변하지 않는 상태로 유지
	 동적바인딩 - 실행 시간에 이뤄지거나 실행 시간에 변경됨


#화살표 함수와 일반함수의 차이점

function 함수 안에서 this 는 위치에 따라 다름
	-그냥전역 : window
	-전역함수 function : window
	-객체안의 함수 : 객체정보
	-이벤트안에서 : 이벤트 발생요소
화살표함수 안에서 this는 무조건 부모요소의 this 값을 상속
